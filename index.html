Model Assumptions & Caveats

What this tool does

A first-order rocket model for comparing transportation architectures. Useful for identifying which routes have inherent delta-v and mass advantages. Not suitable for mission design.

Simplifications & limitations

  • Per-segment dry mass is modeled as structural mass that is added to the end-mass of each segment and then dropped (staged) before the next segment begins. Set Dry Mass = 0 for a single-stack model.
  • No staging optimization. The model evaluates the dry mass and propellant fraction you specify; it does not optimize stage sizing.
  • Δv values are Hohmann-transfer approximations at mean Earth–Mars distance. Actual values vary ±15–20% depending on launch window alignment.
  • Aerobraking is massless propulsively; the heat-shield mass should be captured by setting the segment Dry Mass field.
  • Cost model: by default, initial mass × $/kg at origin. Per-segment Cost Override replaces this with override × that segment’s initial mass × reuse factor — useful for the Earth→LEO leg, which in reality is a multi-stage launch with a well-known $/kg figure rather than a single-stage rocket equation result.
  • Mars/Deimos-origin routes use the ISRU propellant cost (default $0) for propellant. This assumes propellant is pre-positioned; the actual cost depends on the delivery architecture.
  • LLO→TMI uses patched-conic approximation (~1.6 km/s). This is much less than LEO→TMI because LLO already sits high in Earth’s gravity well — the core argument for using lunar-derived propellant.
  • Lunar refueling cost model (when enabled): Earth launches only the vehicle (payload + all dry mass) + ascent propellant; in-space propellant comes from the Moon at the lunar-prop cost. Per-segment overrides take precedence on the affected segment.

Δ-v reference values

  • Earth surface → LEO: ~9.4 km/s (includes ~1.5 km/s gravity and drag losses)
  • LEO → TMI: ~3.6 km/s (Hohmann, Oberth effect from LEO)
  • LLO → TMI: ~1.6 km/s (patched-conic; the chief lunar-launch advantage)
  • Mars arrival, direct to Deimos: aerocapture ~2.3 km/s, propulsive ~4.2 km/s
  • Mars arrival, direct to surface (EDL): aerocapture ~1.0 km/s, propulsive ~6.3 km/s
  • Deimos ↔ Mars surface: aero ~1.5 km/s, propulsive ~5.5 km/s; surface→Deimos ~5.5 km/s
  • Deimos → TEI: ~1.9 km/s; Mars surface → TEI direct: ~5.5 km/s

Purpose

This tool is designed for comparative trade studies only. The primary question is whether Deimos provides logistical leverage as a staging node relative to direct Earth–Mars transport. Compare routes relative to each other; do not treat absolute values as precise mission parameters.

Key insight to watch for

Deimos sits near the top of Mars’ gravity well. The question is whether pre-positioning propellant or cargo at Deimos reduces the overall mission mass budget for subsequent surface operations, even accounting for the extra Δv to reach Deimos first — particularly on the return leg, where leaving directly from Deimos rather than ascending from the surface to TEI offers a large delta-v advantage.
